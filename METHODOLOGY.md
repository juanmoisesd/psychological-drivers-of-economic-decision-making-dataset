## Abstract

This dataset compiles global evidence on psychological factors that drive economic decision-making, spanning behavioral economics, cognitive psychology, and neuroeconomics research published from 1990 to 2025. It covers four core psychological variables across four key economic contexts: financial decisions, investment, consumption, and fraud.

PSYCHOLOGICAL VARIABLES AND KEY DATA:
&nbsp;

Risk Aversion: ~1,000+ published studies (1990&ndash;2025). Average sample size: ~200 participants. Main countries: USA, Europe (Germany, UK, Netherlands), Asia, Latin America. Typical age range: 18&nd

## Data Collection Protocol

### Phase 1 — Search
Systematic search conducted in international databases including WHO, PAHO, World Bank,
ECLAC and national institutional repositories. AI-assisted search (Perplexity AI) with
human review and validation of all values.

### Phase 2 — Extraction
Data extracted and harmonized using Python (pandas). Country codes standardized to
ISO 3166-1 alpha-2. Temporal coverage aligned to annual frequency where available.

### Phase 3 — Validation
All values verified against primary sources. Inconsistencies flagged and corrected.
Data quality scores assigned (high/medium/low) based on source reliability and completeness.

### Phase 4 — Publication
Dataset deposited in Zenodo with DOI under CC0 1.0 license.
Metadata registered with DataCite and indexed in OpenAlex.

## Limitations

- Data coverage varies by country and year
- Some values estimated from regional averages
- AI-assisted collection requires verification for critical applications
- Temporal gaps may exist in lower-income country data
